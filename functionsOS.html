Function Of An Operating system

  1. process management:
    Process management helps OS build processes and delete them. This also offers mechanisms for interposes coordination and communication.
  2. Memory management:
    Memory management module performs the function of allocating and de-allocating memory space to programs that require it.
  3. File management:
    It handles all the file-related activities such as storing, retrieving, naming, sharing, and file security.
  4. Device Management:
    It keeps track of all apps. This module is also known as the I / O controller which is responsible for this function. This also performs the function of allocating machines and de-allocating them.
  5. I / O Device Management:
    One of the main objects of any OS is to hide from the user the peculiarities of that hardware system.
